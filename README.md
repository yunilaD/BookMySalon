# Salon Booking Management System

A web-based appointment booking platform for salons that allows customers to create accounts, log in, and schedule appointments online.

## Features
- User registration and login authentication
- Book salon appointments
- View and manage scheduled bookings
- Responsive user interface
- Secure data storage using a database

## Technologies Used
- Frontend: HTML, CSS, JavaScript (with some React components)
- Backend: Node.js, Express.js
- Database: MongoDB
- Deployment: Cloudflare (Frontend), Railway (Backend)

## How It Works
1. Users create an account or log in
2. Customers select an available time slot
3. The appointment is saved to the database
4. Users can view or manage their bookings
